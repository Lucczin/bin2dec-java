# Bin2Dec - Java

Projeto simples que converte números binários para decimal.

##  Sobre
Este projeto foi desenvolvido como prática de lógica de programação em Java.

##  Funcionalidades
- Entrada de número binário
- Validação (apenas 0 e 1)
- Conversão para decimal
- Loop para múltiplas execuções
